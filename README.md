# SQLBolt Solutions 

This repository contains my documented solutions for SQLBolt SQL exercises.

The purpose of this project is to practice and demonstrate my understanding
of SQL fundamentals through clean, well-structured, and well-documented queries.

These solutions are organized for easy reference and showcase my step-by-step learning process.

## Topics Covered
- Basic SELECT queries
- Queries with constraints (WHERE)
- Filtering and sorting query results
- ORDER BY, LIMIT, and OFFSET
- Multi-table queries with JOINs
- OUTER JOINs
- Working with NULL values
- Queries with expressions
- Aggregate functions (COUNT, SUM, AVG, etc.)
- GROUP BY and HAVING
- Order of execution of SQL queries
- Inserting rows (INSERT)
- Updating rows (UPDATE)
- Deleting rows (DELETE)
- Creating tables (CREATE TABLE)
- Altering tables (ALTER TABLE)
- Dropping tables (DROP TABLE)

## Project Structure
- Each lesson is saved in a separate `.sql` file
- Files are named by lesson number and topic
- Each file includes:
  - Lesson title and goal
  - Exercise tasks
  - SQL solutions
  - Explanations and key takeaways

## Tools Used
- SQL
- SQLBolt
